# Kechow Client Documentation

This directory contains all project documentation for the Kechow client application.

## Documentation Structure

- **[LAYOUTS.md](./LAYOUTS.md)** - Layout components, navigation, and screen mappings
- **[ARCHITECTURE.md](./ARCHITECTURE.md)** - System architecture and design patterns

## Adding Documentation

All project documentation should be stored in this `docs/` folder. When creating new documentation:

1. Create the file in the `docs/` directory
2. Add a link to it in this README
3. Follow the existing documentation style and format
4. Use Spanish for user-facing documentation, English for technical documentation

## Documentation Standards

- Use Markdown format (`.md` files)
- Include clear headings and structure
- Add code examples where relevant
- Keep documentation up-to-date with code changes
